# ROOFTIFY
> Personalized Web Music Player
 
Rooftify는 스웨덴의 음악 스트리밍 서비스인 Spotify의 API를 활용한 Web Music Player 입니다. 사용자가 평소에 듣는 음악을 바탕으로 사용자의 탑 트랙들과 탑 아티스트들의 음악들을 쉽게 선택하여 재생할 수 있는 서비스입니다.
 
## 웹 애플리케이션 링크
  https://rooftify.herokuapp.com/
